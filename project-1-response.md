FISHES:
This magnetic fish game feels very intentional in how it keeps things simple while still being engaging. The visuals are minimal, which works in favor as the focus stays on interaction rather than decoration. The aesthetic is clean and restrained collaborating well with the experience/feel being direct and responsive pointer (mouse) esc.

Visually, it leans into a very raw browser-native look and the interface feels unpolished in a way that reads as stylisticly "unfinished" in a good way. Fish like elements are rendered flat but with nice high-contrast colors and rely on basic shapes (contexts) and said simple color relationships instead of detailed imagery. This gives it a kind of nostalgic early-web or prototypical feel, where the materiality of HTML, CSS, and the canvas is felt visible (without seeing).

There is a strong sense of spatial emptiness. Large areas of negative space isolate the interactive elements, which makes every movement or change feel more pronounced. As I interact with these dynamic colorfull fish-forms it promotes liveliness (it stands out immediately) for the visual field is otherwise so quiet.

FILTERS:
Truly this filter-effect game(?) produces clear emphasis on the heavily stylized "atmosphere" (early 2000s) and composition. Its interface feels well designed and exposed likewise of the innards of a windows PC with elements arranged in that way to guide attention across the screen (it reminds me of the bouncing DVD logos).
There is a stronger use of color, which helps establish this distinct visual identity and mood. Besides the UI and menu (visual aesthetics) commentary, the project presents itself as an interactive visual face filter (likwise of basic apple filters?), and again its aesthetic is built around transformation of a certain static (feeling) composition. The filter processes and alters of the image or video input (you) in real time continue to shape core visual languages discussed, making "effects" themselves the primary aesthetic element emphasized.
Exists externally, also an emphasis on immediacy as such our feedback loop between input and output is so direct, this is in order to clearly perceive how our presence or actions influence the visual result, feeling slightly performative as we pose for the camera.

CLASS-NOTES:
An important project that reads as a direct synthesis of CART 263 class material, but filtered through a personal aesthetic and slightly more experimental framing. A direct embodiment of “media as material” ideas from class notes, but pushed into a more expressive, almost playful filter system.

As for the concept, it stays very close to the structure of (IDEA + both) the code itself. The notes menu options have a raw, instructional quality, especially the pixelly font. The layout is simple, highlighting the ideas of default HTML elements, minimal styling, and clear separation of components. It feels transparent, where you can almost “read” how it works just by looking at it??

I feel a great comfort in how playful this feels in a very functional way, almost like an interactive notebook where class concepts are turned into something you can test and explore...
